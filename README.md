# 🏦 Loan Approval Prediction ML Pipeline

## 📌 Project Overview
This project implements an end-to-end supervised machine learning pipeline to predict loan approval status. It includes data preprocessing, handling missing values, encoding categorical variables, and feature engineering to improve model performance. The dataset is split into training and testing sets and scaled using standardization techniques. Multiple classification models are trained and compared. The performance of each model is evaluated using accuracy, precision, and recall.

---

## ⚙️ Installation & Usage

Follow these steps to run the project locally:

1. Install Python (if not already installed)

2. Clone this repository  
git clone https://github.com/bazik-0/Loan-Approval-ML-Pipeline.git  

3. Navigate into the project folder  
cd Loan-Approval-ML-Pipeline  

4. Install required dependencies  
pip install -r requirements.txt  

5. Run the project  
python loan_approval_ml.py

## 🔧 Features
- Data cleaning and missing value handling  
- Categorical encoding (Label Encoding and One-Hot Encoding)  
- Feature engineering  
- Feature scaling using StandardScaler  
- Implementation of multiple machine learning models  
- Evaluation using accuracy, precision, and recall  

---

## 📊 Models Used
- Logistic Regression  
- Naive Bayes  
- K-Nearest Neighbors (KNN)  

---

## 🎯 Objective
To build and compare different machine learning models for predicting loan approval and analyze their performance using classification metrics.
